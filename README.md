# BSTS301 – Advanced Competitive Programming (ACP) – 01

### Java Programs Repository

This repository contains all Java programs practiced and taught in the **BSTS301 – Advanced Competitive Programming – 01** course.

These programs cover a wide range of algorithmic topics essential for competitive programming, including:

## 🧩 Topics Covered

### **1. Math-Based Algorithms**

- Simple Sieve
- Segmented Sieve
- Euler’s Totient (φ)
- Strobogrammatic Number
- Chinese Remainder Theorem
- Toggle Switch (Alice Apple Tree)

### **2. Bitwise Algorithms**

- Binary Palindrome
- Booth’s Multiplication
- Euclid’s GCD
- Karatsuba Multiplication
- Longest Sequence of 1s (flip a bit)
- Swap Two Nibbles

### **3. Arrays & Searching**

- Block Swap (Array Rotation)
- Maximum Product Subarray
- Maximum Hourglass Sum
- Maximum Equilibrium Sum
- Leaders in Array
- Majority Element (Moore Voting)

### **4. Sorting & Strings**

- Lexicographically First Palindromic String
- Natural Sort Order
- Quick Sort, Selection Sort
- Weighted Substring
- Move Hyphens to Beginning
- Manacher’s Algorithm (Longest Palindromic Substring)

### **5. Recursion & Backtracking**

- Sorted Unique Permutations
- Maneuvering (Grid DFS/BFS)
- Combinations
- Josephus Problem
- Maze Solving
- N-Queens Problem
